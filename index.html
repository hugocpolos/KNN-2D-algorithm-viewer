Visualizador do algoritmo de aprendizado de máquina KNN.

Insira dados com o mouse, no modo True, False ou Learn.
O modo Learn deduzirá o valor do dado de acordo com o algoritmo KNN.
O modo de Flood inserirá dados aleatórios no grid, deduzindo seu valor.

Instruções:
Z - altera o fator K.
X - Altera o modo de inserção de dados.
S - Inicia / Para o flooding de dados.
R - Reinicia o Programa.
